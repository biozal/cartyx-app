Key UI Behaviors

Tag Filter (NotesFilterWidget)

  • New row below session/visibility filters
  • Chip-based input, same style as NoteModal
  • Autocomplete dropdown on focus/type
  • Selected tags filter with AND logic
  • Removing a chip re-runs the query

Tag Autocomplete (NoteModal)

  • Enhances existing tag chip input
  • Dropdown shows matching tags on keystroke
  • Keyboard nav: arrow keys + Enter to select
  • Click to select from dropdown
  • Already-added tags hidden from suggestions
  • New tags auto-created in Tag collection on save
