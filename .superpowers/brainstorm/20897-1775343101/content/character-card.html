Key Details

  • Avatar fallback: When no picture uploaded, show initials on a gradient background (deterministic color from name hash)
  • Uploaded picture: Replaces the initials with the cropped circular image
  • Info line: Race · Age · Location — only shows fields that have values (Grix has no age/location)
  • Visibility icon: Globe (green) for public, Lock (amber) for private — same as Notes
  • Tags: Colored pills, same component as Notes
  • Draggable: Same drag payload pattern as Notes — { collection: 'character', documentId, title }
  • Click: Opens CharacterModal in edit mode
