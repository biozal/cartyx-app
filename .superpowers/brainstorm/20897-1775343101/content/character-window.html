Details

Theron is a wandering half-elf ranger who has spent the last century mapping the uncharted forests of the Western Reaches. He serves as a reluctant guide for adventurers passing through Silverdeep, though his true purpose is guarding the ancient grove hidden deep within the forest.

He carries a longbow carved from a treant's gift and speaks fluent Sylvan. His familiar, a silver hawk named Zephyr, scouts ahead during travels.

GM Notes

Key Details

  • Title bar: Provided by FloatingWindow — adds edit pencil icon next to the minimize/maximize/close buttons
  • Edit icon: Opens CharacterModal in edit mode, same form as wiki panel edit
  • Portrait: Circular cropped image (or initials fallback), centered at top
  • Stats grid: 2-column grid, only shows fields that have values
  • Details accordion: Public notes rendered as markdown, open by default
  • GM Notes accordion: GM-only notes rendered as markdown, collapsed by default
  • Scrollable: Content area scrolls if character has long notes
  • Optimistic updates: Editing from the window updates content in real-time after save
